# Smart Task Manager
A modern task management web app built with React.js featuring dark mode, animated gradient background, and glassmorphism UI.

## Features
- Add, delete, and complete tasks  
- Filter tasks (All / Active / Completed)  
- Dark mode toggle with persistent theme  
- Animated gradient background  
- Glassmorphism UI  
- Data persistence using LocalStorage  

## Technologies
- React.js  
- JavaScript  
- HTML  
- CSS
